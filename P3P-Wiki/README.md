# Persona 3 Portable — SEES Archive 🎴

Fan wiki for P3P — characters, guides, party builder, and full Dark Hour aesthetic.

## ✨ Sections
- **Characters** — 14 cards with filters, search, favorites, modal + stats radar
- **All Characters** — full expanded dossier view: portrait, bio, all stats, unlock guide, skills
- **About** — game lore and mechanics
- **Class Guide** — all school Q&A by month + interactive quiz mode
- **Tartarus** — all 6 blocks: floors, bosses, barriers, safe rooms, strategies
- **Party Builder** — pick 3 members, see element coverage + synergy rating

## 🗂️ File Structure
```
P3P-Wiki/
├── index.html
├── css/styles.css
├── js/
│   ├── script.js             ← Data + UI + Class guide + Quiz
│   ├── effects.js            ← GLSL shader, cursor, idle overlay
│   ├── optimizations.js      ← Preloader, lazy load, visibility API
│   ├── cookies.js            ← Consent banner, favorites, state memory
│   ├── new-effects.js        ← Scroll reveal, constellation, neon flicker,
│   │                            typing effect, URL sync, share button
│   ├── premium-effects.js    ← 3D tilt+shine, section wipe, blood moon,
│   │                            stats radar chart in modal
│   └── content-sections.js  ← Tartarus guide, Party Builder,
│                                All Characters full dossier
└── assets/images/characters/  ← 14 portraits
```

## 🚀 Run
Open `index.html` — no build tools needed.
GitHub Pages: Settings → Pages → main / root.
